**Версия:** 1.0  
**Дата:** 22.02.2025  
**Цель:** Превратить MVP в полноценное приложение уровня production с профессиональным UI/UX и расширенной интеграцией с Telegram-ботом.

## 1. Обзор проекта

**МГИМО AI** — Telegram Mini App для изучения академической лексики (дипломатия, право, экономика, политология). Стек: React 19, Vite, Tailwind, Express, SQLite, Google Gemini API.

### Текущее состояние
- Функциональный MVP с базовым UI
- Визуально «сухой», минималистичный интерфейс
- Бот: /start, /app, /tip, /remind — без доступа к словарю

### Целевое состояние
- Приложение, выглядящее как продукт опытной команды
- Плавные анимации, микро-взаимодействия, визуальная иерархия
- Бот присылает слова пользователю по команде

## 3. Блок 2: Telegram-бот — Присылать слова

### 3.1 Постановка задачи

Пользователь пишет боту команду — получает 1–5 случайных слов из общего словаря (или по теме). Слова отображаются в читаемом формате с переводом и примером.

### 3.2 Требования

| ID | Требование | Приоритет |
|----|------------|-----------|
| B1 | Команда `/word` — присылает 1 случайное слово | P0 |
| B2 | Команда `/words N` — присылает N слов (1–5, по умолчанию 3) | P0 |
| B3 | Опционально: `/words тема` — слова по категории (Международное право, Дипломатия и т.д.) | P1 |
| B4 | Формат сообщения: слово, транскрипция, перевод, пример | P0 |
| B5 | Кнопка «Открыть приложение» под каждым словом | P1 |

### 3.3 Архитектура

```
User → /word → Telegram Bot → Backend API (GET /api/words/random?count=1)
                                    ↓
                              SQLite (words)
                                    ↓
User ← Форматированное сообщение ← Bot
```

**Зависимости:**
- Бот должен иметь доступ к Backend API (переменная `API_URL`)
- Backend: новый endpoint `GET /api/words/random?count=N&category_id=...`

### 3.4 API Backend

**Endpoint:** `GET /api/words/random`

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| count | number | нет (default: 3) | Количество слов (1–10) |
| category_id | number | нет | ID категории для фильтра |

**Ответ:** `{ words: [{ id, word, translation, transcription, example, example_translation, category_name }] }`

### 3.5 Формат сообщения бота

```
📖 Слово дня: Sovereignty

/ˈsɒvrɪnti/
Суверенитет

Пример: "The nation fought hard to maintain its political sovereignty."
"Нация боролась за сохранение своего политического суверенитета."

[📖 Открыть приложение]
```

### 3.6 Задачи реализации

- [x] Backend: `GET /api/words/random`
- [x] Bot: обработка `/word`, `/words`, `/words N`, `/words тема`
- [x] Bot: переменная `API_URL` в .env
- [x] setup-telegram-bot: добавить команду `word` в setMyCommands
- [x] Документация: обновить README, DEPLOYMENT

## 7. Риски и ограничения

| Риск | Митигация |
|------|-----------|
| Бот и Backend на разных хостах | API_URL в env бота, CORS не нужен (server-to-server) |

| Rate limit Telegram API | Стандартные лимиты, без массовой рассылки |
